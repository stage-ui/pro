STAGE Website - Complete Wireframe

Study of Theatre Arts & Global Entertainment | Production Media, Vocational Program, University of Indonesia

Multi-page Static Website with Gamification System

Project Overview

STAGE is a gamified educational website for the Performing Arts concentration at University of Indonesia's Vocational Program. It features an XP-based progression system, missions, rewards, and video performances.

Target Users: Students interested in performing arts, theatre, and stage performance
Core Features: Gamification (XP, Levels, Missions, Rewards), Video Gallery, Program Information
Authentication: Session-based login system

Responsive Design Specifications

Breakpoint Width Layout Changes
Mobile < 768px • Single column grids
• Reduced padding (20px → 15px)
• Footer: 1 column
• Hero illustration: 400px → 320px
• Video frame: 380px → 320px
• Products grid: 1 column (<480px)
Tablet 768px - 1024px • 2 column grids where applicable
• Comment panel: 380px → 320px
• Standard padding maintained
Desktop > 1024px • Full multi-column layouts
• Max-width: 1200px content
• All features visible
